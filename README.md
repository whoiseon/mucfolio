##Introduce.

<b>먹폴리오(mucfolio)</b>노션에서 영감을 받아 진행한 프로젝트 진행 기록 어플리케이션입니다.</br>
먹폴리오에서는 프로젝트를 폴더처럼 간편하게 생성하여 그 하위 폴더로 여러가지 스케줄을 기록하며</br>
나의 프로젝트 진행상황을 스텝별로 간단하게 기록할 수 있습니다. 또 빠른 메모를 통해 간단하게 </br>
프로젝트를 별개로 메모를 기록할 수 있습니다.
